Previous Projects

Inter-Prediction of H.264/AVC Encoder

I designed in VHDL a Inter-Prediction part of H.264/AVC Encoder at SIS Lab. This module uses reference frame and current frame of video to find out the relationship between them. Normally, in current frame, some objects moved and some object and background stay as same as reference frames. As finding this motion (standardize as motion vector with pixel or sub-pixel unit) and just compress the difference value, encoder can obtain significant smaller bit-rate. This work is published in a paper.
